# Sneha Maiti — Portfolio

Personal portfolio site. AI Marketing Operator and Growth Strategist.

This is a single static file. `index.html` contains all the markup, styles, scripts, and the embedded photo. There is no build step.

## Deploy

Drop `index.html` into any static host. On Vercel, import the repo and set the framework preset to Other with empty build settings. The site serves `index.html` automatically.

## Local preview

Open `index.html` in a browser, or run a simple server:

```
python3 -m http.server 8000
```

Then visit http://localhost:8000
